WAMA : Webapp for Automatic Media Anonymization

WAMA detects and blurs faces and number plates quickly and accurately.

How To Use It?


Upload medias

  • Photos
  • Videos
  • Media folder

Define options

  • What to blur?
    • Faces (True/False)
    • Plates (True/False)
    • People (True/False)
    • Cars (True/False)
  • How to blur?
    • blur size (blur size value)
    • detection threshold (detection threshold value)
    • ROI enlargement (ROI enlargement value)
  • What to show?
    • boxes (True/False)
    • show labels (True/False)
    • show conf (True/False)
    • show (True/False)
