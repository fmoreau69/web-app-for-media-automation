About WAMA and RGPD compliance

WAMA detects and blurs faces, number plates and so much more, quickly and accurately.

Why ?

  • Needs:
    • Blur the videos of Lescot's archived experimental data
    • To meet the requirements of the RGPD and the CNIL in terms of the retention and use of media (photos, videos).
  • Objectives:
    • To be able to keep the lab's video data provided it is anonymised
    • To have at Lescot, and then at the UGE, an online service for automatic anonymisation of media
    • To be able to integrate this tool into future research projects

What is it?

  • A website using algorithms to detect and blur objects in images
  • Client-server website: User settings are sent to the server then processing is carried out on the server by applying these parameters
  • Detection algorithms based on deep learning principles: Creation/collection of data sets => Training => Detection model

How does it work?

  • Pytorch: Deep learning framework initially developed by Facebook's AI research group released as open source on GitHub in 2017
  • Yolov8: You Only Look Once - Real-time object detection framework
