WAMA: Web App for Media Automation

WAMA, Web App for Media Automation, is a web application developed at Lescot and based on deep learning, offering AI tools applied to media.

    Planned developments include (See the GitHub project) :
  • Anonymizer: automatic blurring of objects in photos/videos
  • Enhancer: improved resolution and quality of photos/videos (in progress)
  • Imager: automatic image generation based on prompt text (in progress)
  • Transcriber: automatic transcription of audio files into text (in progress)
  • Synthesizer: automatic voice synthesis from text files (in progress)
