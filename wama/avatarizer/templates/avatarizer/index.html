À propos de l'Avatarizer

Qu'est-ce que l'Avatarizer ?

L'Avatarizer génère des vidéos d'avatars animés synchronisés avec la parole, idéales pour créer des consignes vidéo standardisées et engageantes pour les expérimentations au Lescot.

Modes de fonctionnement
  • Pipeline : saisissez un texte → le service TTS génère l'audio → MuseTalk anime l'avatar
  • Standalone : uploadez votre propre fichier audio → MuseTalk anime l'avatar
Avatars
  • Galerie partagée : avatars curatés par l'administrateur dans media/avatarizer/gallery/
  • Image personnalisée : importez n'importe quelle photo (JPG, PNG)
Technologie

Basé sur MuseTalk v1.5 (Tencent Music, MIT) : synchronisation labiale temps réel à partir d'une image portrait et d'un signal audio. Amélioration optionnelle par CodeFormer (restauration faciale haute qualité).

Guide d'utilisation

Mode Pipeline (texte → TTS → avatar)
  1. Sélectionnez le mode Pipeline dans le panneau de droite
  2. Saisissez le texte de la consigne
  3. Choisissez le modèle TTS, la langue et la voix
  4. Sélectionnez un avatar dans la galerie ou importez une photo
  5. Cliquez sur Générer la vidéo
Mode Standalone (audio existant)
  1. Sélectionnez le mode Standalone
  2. Glissez ou sélectionnez un fichier audio (WAV, MP3…)
  3. Sélectionnez un avatar
  4. Cliquez sur Générer la vidéo
Paramètres MuseTalk
  • Mode Rapide : MuseTalk seul — temps réel, 6–8 GB VRAM
  • Mode Qualité : MuseTalk + CodeFormer — meilleure netteté faciale, légèrement plus lent
  • Décalage visage : si la zone de la bouche est mal détectée, ajustez de ±1 à ±5
  • Amélioration CodeFormer : restauration faciale haute qualité (actif en mode Qualité)
