Guide d'utilisation

Image & Vidéo — Upscaling
  • Glissez-déposez ou cliquez sur "Parcourir" pour sélectionner vos fichiers
  • Choisissez le modèle AI (vitesse vs qualité) et configurez le débruitage
  • Cliquez sur par fichier ou "Démarrer" pour tout lancer
  • Téléchargez avec ou "Télécharger tout" (ZIP)
Astuce : Pour les vidéos, utilisez RealESRGANx4 + débruitage. Pour la vitesse, préférez RealESR_Gx4.
Audio — Speech Enhancement
  1. Passez en mode Audio avec le bouton en haut du panneau droit
  2. Glissez-déposez vos fichiers audio (MP3, WAV, FLAC…) ou cliquez "Parcourir"
  3. Choisissez le moteur et les paramètres :
  • Resemble Enhance (recommandé) : sélectionnez le mode et la force du débruitage
  • DeepFilterNet 3 : rapide, pas de paramètre supplémentaire
  1. Cliquez sur ou "Démarrer" pour lancer
  2. Téléchargez l'audio amélioré en WAV
Quel moteur choisir ?
  • Resemble Enhance : qualité maximale, remplace Adobe Podcast — mode "Débruitage + Amélioration" recommandé
  • DeepFilterNet 3 : ultra-rapide (<1 GB VRAM), idéal pour traiter de nombreux fichiers rapidement
Installation requise :
pip install resemble-enhance soundfile (Resemble)
pip install deepfilternet (DeepFilterNet)
