À propos de Reader

Qu'est-ce que Reader ?

Reader extrait automatiquement le texte de vos documents numérisés, photos et PDFs grâce à des moteurs OCR open source de dernière génération. Supporte le texte imprimé, les notes manuscrites, les tableaux et les formules.

Moteurs OCR disponibles
olmOCR-2 7B Principal

Allen AI — VLM dédié documents — imprimé + manuscrit + tableaux + formules — ~14 Go VRAM

docTR CPU-friendly

Mindee — pipeline PyTorch — imprimé, formulaires, reçus — fonctionne sans GPU

Formats supportés

PDF (multi-pages), JPG, PNG, TIFF, WebP, BMP

Technologie

olmOCR-2 (Allen AI / Qwen2-VL fine-tuné) — docTR (Mindee, PyTorch) — pymupdf / pdf2image pour conversion PDF

Guide d'utilisation

Importer des documents
  • Glissez-déposez vos fichiers ou cliquez sur la zone d'import
  • Formats : PDF (multi-pages), JPG, PNG, TIFF, WebP, BMP
  • Plusieurs fichiers à la fois — traitement en file d'attente
Paramètres
  • Moteur : Auto sélectionne olmOCR-2 si le GPU est disponible, sinon docTR
  • Mode : Auto (recommandé), Imprimé, Manuscrit
  • Format de sortie : Texte brut (.txt) ou Markdown (.md)
  • Langue : Laisser vide pour auto-détection
Traitement
  • Cliquez Démarrer sur un item ou Démarrer tout
  • Statuts : En attente → En cours → Terminé
  • Aperçu du texte extrait visible directement dans la carte
Quel moteur choisir ?
  • olmOCR-2 : Notes manuscrites, documents complexes, tableaux, formules
  • docTR : Documents imprimés propres, formulaires, pas de GPU disponible
