À propos de Converter

Qu'est-ce que Converter ?

Converter change le format de vos fichiers — images, vidéos, audio, documents et archives — sans perte de qualité inutile. Tout est converti localement sur votre serveur WAMA : aucun fichier n'est envoyé vers un service tiers.

Aide — Converter

Convertir un fichier
  1. Déposez un fichier dans la zone prévue (glisser-déposer) ou cliquez pour le sélectionner. Vous pouvez aussi l'envoyer depuis le Gestionnaire de fichiers (« Envoyer vers → Converter »).
  2. Le format de sortie est proposé automatiquement selon le type ; ajustez-le si besoin via le bouton Paramètres.
  3. Choisissez la qualité (Web / Équilibré / Maximum).
  4. Cliquez sur Démarrer, puis Télécharger une fois terminé.
Traitement par lot
  • Déposez plusieurs fichiers en même temps : ils sont regroupés automatiquement en lots par nature (images, vidéos, documents…).
  • Ou importez un fichier liste (.txt / .csv) contenant des chemins ou URLs, un par ligne.
  • Réglez le format/qualité du lot via dans l'en-tête du groupe, puis pour tout lancer.
Profils

Enregistrez une combinaison format + qualité + options comme profil pour la réappliquer en un clic sur vos prochaines conversions.

Bon à savoir
  • PDF → DOCX : la mise en forme, les images et les tableaux sont reconstruits. Un PDF entièrement scanné (image) donnera un DOCX avec les images mais sans texte sélectionnable — utilisez le Reader (OCR) pour extraire le texte.
  • Toutes les conversions sont locales : vos fichiers ne quittent jamais le serveur.
