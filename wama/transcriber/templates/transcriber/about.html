About - WAMA Transcriber

Automatic transcription of audio files into text.

What is it?

  • An interface to upload audio and obtain text transcripts
  • Backed by speech_to_text_transcriptor or Whisper as fallback

How does it work?

  • Upload an audio file and start processing
  • Follow progress and download the text when finished
