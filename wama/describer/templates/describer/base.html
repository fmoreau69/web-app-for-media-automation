À propos de Describer

Qu'est-ce que Describer ?

Describer analyse et synthétise automatiquement vos contenus multimédia. Il décrit des images, résume des textes et PDF, transcrit des audios et analyse des vidéos — le tout intégré à WAMA.

Moteurs d'analyse
Qwen3.5 9B (Ollama)

Résumé, analyse, compte-rendu — texte, PDF, DOCX

Moondream2 (Ollama)

Description d'images — 1.8B multimodal léger

Whisper (OpenAI)

Transcription audio — 99 langues

BLIP (fallback)

Description d'images locale si Ollama indisponible

Fonctionnalités
  • Images : Description visuelle avec Moondream2 (Ollama) → BLIP (fallback)
  • Textes / PDF / DOCX : Résumé et analyse avec Qwen3.5 9B
  • Audios : Transcription Whisper + résumé LLM
  • Vidéos : Extraction de frames + description + transcription audio
  • Compte-rendu de réunion : Format structuré (résumé, décisions, actions)
Formats de sortie
  • Résumé court : Synthèse concise en 2-3 phrases
  • Description détaillée : Analyse complète avec points clés
  • Synthèse scientifique : Format académique structuré
  • Points clés : Liste à puces
  • Compte-rendu de réunion : Résumé exécutif, décisions, actions à mener
Technologie

Analyse LLM : Qwen3.5 9B via Ollama — Vision : Moondream2 (Ollama) / BLIP (local) — Audio : Whisper OpenAI.

Guide d'utilisation

Téléchargement de fichiers
  • Glissez-déposez vos fichiers dans la zone de téléchargement
  • Ou cliquez sur "Parcourir" pour sélectionner manuellement
  • Formats supportés : Images (JPG, PNG, WebP), Vidéos, Audio, PDF, DOCX, TXT, MD
  • Le type de contenu est détecté automatiquement
Options de description
  • Format : Résumé court, détaillé, scientifique, points clés, ou compte-rendu de réunion
  • Langue : Français ou Anglais — la réponse LLM suit la langue choisie
  • Longueur : Longueur maximale du résumé (pour les textes courts déjà concis)
Lancer le traitement
  • Cliquez sur "Démarrer" pour lancer l'analyse
  • Statut : En attente → En cours → Terminé
  • Le résultat apparaît dans la colonne "Description"
Télécharger les résultats
  • Téléchargez chaque résultat en fichier texte
  • Utilisez "Télécharger tout" pour un ZIP de tous les résultats
Quel format choisir ?
  • Résumé court : Idéal pour un aperçu rapide (2-3 phrases)
  • Points clés : Liste à puces des informations essentielles
  • Compte-rendu de réunion : Résumé exécutif + décisions + actions à mener (LLM)
  • Synthèse scientifique : Pour articles et publications académiques
