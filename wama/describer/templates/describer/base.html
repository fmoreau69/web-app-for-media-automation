A propos de Describer

Qu'est-ce que Describer ?

Describer est une application d'analyse et de synthese de contenus multimedia integree a WAMA. Elle utilise des modeles d'IA pour decrire des images, resumer des textes, transcrire des audios et analyser des videos.

Fonctionnalites
  • Images : Description visuelle automatique avec BLIP
  • Videos : Analyse de scenes + transcription audio
  • Audios : Transcription avec Whisper + resume
  • Textes/PDF : Resume et synthese avec BART/Pegasus
  • Publications scientifiques : Extraction des points cles
Formats de sortie
  • Resume court : Synthese concise
  • Description detaillee : Analyse complete
  • Synthese scientifique : Format academique
  • Points cles : Liste a puces
Technologie

Propulse par des modeles open-source : BLIP pour les images, BART/Pegasus pour le resume de texte, et Whisper pour la transcription audio.

Guide d'utilisation

Telecharger des fichiers
  • Glissez-deposez vos fichiers dans la zone de telechargement
  • Ou cliquez sur "Parcourir" pour selectionner manuellement
  • Formats supportes : Images, Videos, Audio, PDF, DOCX, TXT, MD
  • Le type de contenu est detecte automatiquement
Options de description
  • Format : Choisissez entre resume, detaille, scientifique ou points cles
  • Langue : Francais, Anglais ou detection automatique
  • Longueur : Ajustez la longueur maximale du resume
Lancer le traitement
  • Cliquez sur "Demarrer" pour lancer la description
  • Suivez la progression en temps reel
  • Le resultat apparait dans la colonne "Description"
Telecharger les resultats
  • Telechargez le resultat au format texte
  • Utilisez "Telecharger tout" pour un ZIP de tous les resultats
Astuce

Pour les publications scientifiques, utilisez le format "Synthese scientifique" pour obtenir une structure academique avec les points cles.
