À propos de Synthesizer

Qu'est-ce que Synthesizer ?

Synthesizer transforme automatiquement du texte en audio de haute qualité, intégré à WAMA. Il propose plusieurs moteurs TTS selon vos besoins : qualité maximale, clonage de voix ou expressivité émotionnelle.

Moteurs TTS disponibles
Higgs Audio V2 RECOMMANDÉ

Voix ultra-réaliste — 8 langues — multi-locuteurs — RTX 4090

XTTS v2 (Coqui)

Clonage de voix — 17 langues — 6s de référence audio

Bark (Suno AI)

Expressif, émotions, rires, silences — multi-langue

Mode multi-locuteurs

Higgs Audio V2 — dialogue naturel entre plusieurs voix

Fonctionnalités
  • Saisie de texte directe ou upload de documents (TXT, PDF, DOCX, CSV, MD)
  • Clonage de voix avec XTTS v2 (échantillon de référence 6-10 secondes)
  • Mode multi-locuteurs avec Higgs Audio V2 (dialogue, scène décrite)
  • Préréglages de voix : par genre, accent, style
  • Ajustement de la vitesse et de la tonalité
  • Prévisualisation audio intégrée avant téléchargement
  • Traitement par lots, téléchargement WAV/MP3
  • Langue par défaut depuis votre profil utilisateur
Technologie

Higgs Audio V2 (Boson AI) — XTTS v2 (Coqui TTS) — Bark (Suno AI) — Microservice FastAPI dédié (port 8001) avec préchargement GPU.

Guide d'utilisation

Paramètres de synthèse
  • Modèle TTS : Higgs Audio V2 (qualité max), XTTS v2 (clonage de voix), Bark (expressif)
  • Langue : Pré-sélectionnée depuis votre profil utilisateur, modifiable par synthèse
  • Préréglage de voix : Choix par genre, accent ou style selon le moteur
  • Vitesse / Tonalité : Curseurs ajustables, réinitialisables
  • Échantillon vocal : Uploadez un fichier audio pour cloner une voix (XTTS v2)
Téléchargement de fichiers texte
  • Glissez-déposez vos fichiers texte dans la zone de téléchargement
  • Ou cliquez sur "Parcourir" pour sélectionner manuellement
  • Formats supportés : TXT, PDF, DOCX, CSV, MD
  • Chaque fichier apparaît dans la file de synthèse
Démarrer le traitement
  • Cliquez sur "Démarrer le traitement" pour lancer la synthèse
  • Traitez les éléments individuellement ou toute la file d'un coup
  • Statut : En attente → En cours → Succès / Échec
  • Barre de progression visible pour chaque synthèse
Prévisualisation et téléchargement
  • Lecteur audio intégré disponible dès la fin de la synthèse
  • Téléchargez l'audio généré au format WAV
Quel moteur choisir ?
  • Higgs Audio V2 : Voix ultra-réaliste, multi-locuteurs, 8 langues — choix par défaut
  • XTTS v2 : Clonage de voix avec 6-10s d'échantillon, 17 langues
  • Bark : Très expressif — rires, pauses, émotions — moins déterministe
