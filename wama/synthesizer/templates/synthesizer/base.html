À propos de Synthesizer

Qu'est-ce que Synthesizer ?

Synthesizer est une application de synthèse vocale intégrée à WAMA qui transforme automatiquement du texte en audio de haute qualité.

Fonctionnalités
  • Interface pour saisir ou télécharger du texte et générer de l'audio
  • Support de text_to_speech_synthesizer avec moteurs de secours
  • Modèle XTTS v2 pour le clonage de voix
  • Choix de la langue et des préréglages de voix
  • Ajustement de la vitesse et de la tonalité
  • Upload optionnel d'échantillon audio pour clonage de voix
  • Support de multiples formats : TXT, PDF, DOCX, CSV, MD
  • Prévisualisation audio intégrée
  • Traitement par lots
Comment ça fonctionne ?
  1. Entrez du texte ou téléchargez un document
  2. Choisissez les options de langue et de voix
  3. Lancez la synthèse
  4. Suivez la progression du traitement
  5. Téléchargez le fichier audio généré
Technologie

Propulsé par des moteurs TTS avancés avec support du clonage de voix pour une qualité audio réaliste et naturelle.

Guide d'utilisation

Options de synthèse
  • Sélectionnez un modèle TTS (XTTS v2 recommandé pour le clonage de voix)
  • Choisissez la langue et le préréglage de voix pour la parole générée
  • Ajustez la vitesse et la tonalité avec les curseurs
  • Réinitialisez tous les paramètres avec le bouton reset
  • (Optionnel) Téléchargez un court échantillon audio pour le clonage de voix
Téléchargement de fichiers texte
  • Glissez-déposez vos fichiers texte dans la zone de téléchargement
  • Ou cliquez sur "Parcourir" pour sélectionner manuellement
  • Formats supportés : TXT, PDF, DOCX, CSV, MD
  • Chaque fichier téléchargé apparaît dans la file de synthèse
Démarrer le traitement
  • Cliquez sur "Démarrer le traitement" pour lancer la synthèse
  • Vous pouvez démarrer les éléments individuellement ou traiter toute la file
  • Mises à jour du statut : En attente → En cours → Succès / Échec
  • Une barre de progression indique l'avancement de la synthèse
Prévisualisation et téléchargement
  • Quand la synthèse est terminée, un bouton Prévisualiser devient disponible
  • Utilisez le lecteur audio intégré pour écouter instantanément
  • Cliquez sur Télécharger pour sauvegarder le fichier audio généré
Gestion de la file
  • Le badge affiche le nombre d'éléments actuellement dans la file
  • Supprimez les éléments individuellement ou utilisez "Tout effacer"
  • Chaque entrée affiche les options de synthèse et les propriétés
Astuce

Pour un clonage de voix optimal avec XTTS v2, utilisez un échantillon audio de 6-10 secondes avec une voix claire et peu de bruit de fond.
