Help - WAMA Synthesizer

Synthesis Options

  • Select a TTS model (XTTS v2 recommended for voice cloning)
  • Choose the language and voice preset for the generated speech
  • Adjust speed and pitch using the sliders
  • Reset all settings using the reset button
  • (Optional) Upload a short audio sample for voice cloning

Upload Text Files

  • Drag and drop your text files into the upload zone
  • Or click “Browse” to select files manually
  • Supported formats: TXT, PDF, DOCX, CSV, MD
  • Each uploaded file appears in the synthesis queue

Start Processing

  • Click “Start Process” to launch synthesis
  • You may start items individually or process the full queue
  • Status updates through: Pending → Running → Success / Failure
  • A progress bar indicates synthesis progression

Preview and Download

  • When synthesis completes, a Preview button becomes available
  • Use the integrated audio player to listen instantly
  • Click Download to save the generated audio file

Queue Management

  • The badge displays the number of items currently in the queue
  • Remove items individually or use “Clear All” to empty the list
  • Each entry displays synthesis options and properties

Console

  • Displays real-time logs during processing
  • Useful for debugging or monitoring details
  • Toggle console visibility using the Console button
