À propos d'Imager

Qu'est-ce qu'Imager ?

Imager génère des images et vidéos IA à partir de descriptions textuelles, intégré à WAMA. Il utilise Diffusers (Hugging Face) et prend en charge des modèles allant de Stable Diffusion à des architectures 20B paramètres comme Qwen Image 2.

Fonctionnalités Images
  • Text-to-Image : Génération à partir de prompts textuels
  • Image-to-Image : Transformation et variation d'images existantes
  • Image Editing : Édition multi-référence avec Qwen Image Edit
  • Dimensions personnalisables (256px à 2048px — 2K natif avec Qwen/HunyuanImage)
  • Prompts négatifs, seed reproductible, guidance scale
  • Génération par lots depuis fichiers (TXT, JSON, YAML)
  • Modèles spécialisés Logo (Flux LoRA, Redmond, Amazing Logos)
Fonctionnalités Vidéos
  • Text-to-Video : Génération de vidéos à partir de prompts
  • Image-to-Video : Animation d'images statiques (CogVideoX 5B I2V)
  • Support 480p/720p, 8 à 30 fps selon le modèle
  • Durée ajustable (2–6s), export automatique en MP4

Guide d'utilisation

Génération d'images
1. Sélectionner l'onglet "Image"

Choisissez le mode : Text-to-Image, Img2Img, Image Editing (Qwen), Logo, etc.

2. Écrire le prompt

Décrivez l'image souhaitée avec précision. Anglais recommandé pour les modèles SD.

3. Configurer les paramètres

Modèle, résolution (512–2048px), steps, guidance scale.

4. Générer et télécharger

Cliquez sur "Ajouter" puis "Démarrer". Téléchargez une fois terminé.

Paramètres clés
Steps20–30 (test) / 50 (qualité) — Qwen Image : 50 par défaut
GuidanceSD : 7–10 équilibré — Qwen Image : true_cfg_scale 4.0
RésolutionSD : 512–1024px — HunyuanImage / Qwen Image : jusqu'à 2048px
SeedLaisser vide pour aléatoire, ou fixer pour reproduire
Exemple :
"Majestic mountain landscape at golden hour, dramatic clouds, alpine meadow, photorealistic, 8k"
Négatif : "blurry, low quality, cartoon, watermark"
Génération de vidéos
1. Sélectionner l'onglet "Vidéo"

Choisissez Text-to-Video (T2V) ou Image-to-Video (I2V).

2. Choisir le modèle

CogVideoX 5B : T2V polyvalent (21GB)
CogVideoX 5B I2V : Animation d'image (21GB)
LTX-Video 13B Distilled FP8 : Rapide T2V+I2V 24fps (8GB)
LTX-Video 13B Distilled : T2V+I2V 24fps (14GB)
Mochi-1 : Qualité cinématique 30fps (22GB)

3. Configurer la vidéo

Résolution (480p/720p), durée (2–6s), steps (30–50).

4. Patienter et télécharger

5–30 minutes selon le modèle. Export automatique en MP4.
