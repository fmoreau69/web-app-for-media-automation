About Imager

What is Imager?

Imager is an AI-powered image generation application integrated into WAMA (Web App for Media Automation). It uses the powerful imaginAIry library to generate stunning images from text descriptions.

Features

  • Generate images from text prompts using advanced AI models
  • Support for multiple Stable Diffusion models
  • Customizable image dimensions (256px to 1024px)
  • Negative prompts to exclude unwanted elements
  • Adjustable generation steps and guidance scale
  • Optional upscaling for higher resolution
  • Seed-based reproducibility
  • Batch generation (up to 4 images at once)
  • Real-time progress tracking
  • Download individual or multiple images

Available Models

Model Description
OpenJourney v4 Artistic, vibrant images with high quality
Dreamlike Art 2.0 Dreamlike, surreal artistic style
Stable Diffusion 2.1 Latest stable version, balanced quality
Stable Diffusion 1.5 Classic version, fast and reliable

How to Use

  1. Enter a descriptive prompt of the image you want to generate
  2. Optionally add a negative prompt to exclude unwanted elements
  3. Choose your model and adjust parameters (size, steps, guidance)
  4. Click "Add to Queue" to create the generation task
  5. Click "Start" on individual generations or "Start All" for batch processing
  6. Monitor progress in real-time
  7. Download your generated images when complete

Tips for Better Results

  • Be descriptive: Include style, lighting, colors, mood in your prompt
  • Use negative prompts: Exclude "blurry, low quality, distorted" for better results
  • Adjust steps: More steps (50-100) = better quality but slower
  • Guidance scale: 7-12 is usually optimal (higher = follows prompt more strictly)
  • Seed: Use the same seed to reproduce an image with variations

Technology

Powered by imaginAIry, an open-source AI image generation library built on Stable Diffusion technology.

Part of the WAMA ecosystem for media automation and processing.

Note: Image generation is computationally intensive and may take several minutes depending on your hardware and chosen parameters. A GPU is recommended for faster processing.
