Imager Help

Quick Start Guide

1. Create a Generation

Enter your prompt in the text area. Be descriptive about what you want to see.

2. Adjust Parameters

Choose model, size, and generation settings to control output quality and style.

3. Start Generation

Click "Add to Queue" then "Start" to begin generating your image.

4. Download Results

Once complete, download your generated images.

Parameters Explained

Prompt

The main description of what you want to generate. Be specific and descriptive.
Example: "A serene mountain lake at dawn, misty atmosphere, photorealistic"

Negative Prompt

What you DON'T want in the image. Helps avoid unwanted elements.
Example: "blurry, low quality, distorted, ugly, watermark"

Model

The AI model to use. Each has a different style:
• OpenJourney v4: Artistic, vibrant
• Dreamlike Art: Surreal, dreamlike
• SD 2.1: Balanced, versatile
• SD 1.5: Fast, reliable

Width & Height

Output image dimensions. Larger = more detail but slower generation.
• 512x512: Standard, fast
• 768x768: Higher detail
• 1024x1024: Maximum quality (slow)

Steps

Number of diffusion steps. More steps = better quality but slower.
• 20-30: Quick, good for testing
• 50-70: Recommended for quality
• 80-100: Maximum quality (very slow)

Guidance Scale

How strictly to follow your prompt.
• 1-5: Creative, loose interpretation
• 7-10: Balanced (recommended)
• 15-20: Strict adherence to prompt

Seed

Random seed for reproducibility. Use the same seed with the same prompt to get similar results. Leave empty for random.

Number of Images

How many variations to generate (1-4). Each image uses the same parameters but different random variations.

Upscale

Increases resolution of generated image. Results in higher quality but significantly increases generation time.

Troubleshooting

Generation Failed

• Check the console for error messages
• Ensure imaginAIry is properly installed
• Try reducing image size or steps
• Check available disk space

Poor Quality Results

• Increase steps (50-100)
• Adjust guidance scale (7-12 recommended)
• Make prompt more descriptive
• Add quality keywords: "highly detailed", "8k", "professional"
• Use negative prompt to exclude unwanted elements

Slow Generation

• Use smaller image sizes (512x512)
• Reduce number of steps
• Disable upscaling
• Generate fewer images at once
• Use a GPU if available (much faster than CPU)

Frequently Asked Questions

How long does generation take?

Depends on hardware and parameters. With GPU: 10-60 seconds. With CPU: 2-10 minutes.

Can I use the images commercially?

Check the license of the specific model you're using. Most Stable Diffusion models allow commercial use, but verify first.

Why are my images different each time?

Unless you specify a seed, generation is random. Use the same seed with the same parameters to get reproducible results.

What's the best model to use?

It depends on your style preference. OpenJourney v4 is great for artistic images, SD 2.1 for general use, and Dreamlike Art for surreal/dreamlike results.
